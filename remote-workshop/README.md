# 職涯價值觀遠端工作坊頁面

這是一個可部署成靜態網站的職涯價值觀卡牌桌，用來遠端帶領參與者選卡、淘汰與排序。

## 使用方式

本機預覽：

```bash
cd remote-workshop
python3 -m http.server 4173
```

打開：

```text
http://127.0.0.1:4173
```

也可以直接開啟 `index.html`。若要在線上使用，可將整個 `remote-workshop` 資料夾上傳到 GitHub Pages、Netlify、Vercel 或其他靜態網站服務。

## 功能

- 深藍星空絨布質感牌桌
- 抽象圖卡風格價值卡，比例接近 88 x 66 mm
- 第一階段：15 張牌庫與 5 個選擇欄位互動，點選牌庫卡片可加入，點選選擇區卡片可放回牌庫
- 第二階段：5 個保留欄位與 2 個淘汰欄位互動，點選保留卡可進淘汰區，點選淘汰卡可回保留區
- 每張價值卡都有對應意境的抽象示意圖
- 側欄可收合，讓牌桌成為主畫面
- 場次名稱、參與者代稱與本次目標
- 每階段追問提示
- 對話紀錄分類：個案明確表達、主持人觀察、主持人推測、共同洞察
- 自動產生 Markdown 摘要
- 本機自動儲存、摘要複製、JSON 下載

## 目前限制

目前版本沒有後端資料庫，也沒有多人即時同步。它先作為單人互動原型，用來確認牌桌、選卡、淘汰與排序的操作感。

若之後要讓參與者各自在線上選卡並即時同步，可以再接 Firebase、Supabase 或其他即時資料庫。
